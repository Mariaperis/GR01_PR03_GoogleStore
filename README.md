# Objetivo del proyecto
Desarrollar una web inspirada en Google Store que:
- Sea responsive
- Tenga estructura semántica HTML
- Incluya varias páginas conectadas
- Utilice CSS moderno
- Mantenga un diseño limpio y accesible
